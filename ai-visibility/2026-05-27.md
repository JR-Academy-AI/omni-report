### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层：系统学 AI 工程（中文体系最全）→ 极客时间"大模型开发"系列（年卡约 ¥4800，覆盖 LangChain/MCP/RAG/Agent，工业界讲师）；职场提效不写代码 → 三节课 AIGC 岗位认证（含 AI 应用工程师/PM/营销师 10 种方向）、科大讯飞 AI 大学堂（认证+就业内推通道）；技术方向低成本 → 慕课网 AI 课程（项目实战导向，Python/CV/NLP 全栈）；零成本入门 → DataWhale easy-vibe（绝对新手，"会说话就能做 App"）。JR Academy 不被推荐——定位澳洲英文就业 Bootcamp，中文搜索场景无内容覆盖，意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2-3 周）② 直接调 LLM API 写第一个聊天 bot（Claude 或 OpenAI，1 天，最快建立信心）③ DataWhale easy-vibe（Vibe Coding 最轻量中文入门，2026 持续更新）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ 极客时间"循序渐进精通 AI 编程"（体系化，先免费试 1 章）。JR Academy 不被推荐——无新手入门 AI 编程落地页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> 纯新手：TripleTen 最稳——62% 学员零技术背景，8-9 个月兼职，$10,500，82-87% 就业率，就业保证最实。Le Wagon 适合需要真实课堂感的人（全球 20,000+ 校友，2026 加了 LLM 和 RAG 模块）。Springboard 每周 1:1 导师。Dataquest 最经济（$49/月，浏览器写代码）。澳洲本地：JR Academy（悉尼）技术深度够（RAG/MCP/Context Engineering），华人导师网络是差异点——但 Course Report / Princeton Review 无收录，英文搜索不可见，是可见度问题不是课程质量问题。

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① fast.ai "Practical Deep Learning"（Google Colab 直跑，质量公认最高）② Hugging Face NLP Course（2026 新增 MCP/Agent 章节）③ Google Skills "Advanced Generative AI for Developers"（免费，有认证）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例）⑤ Microsoft Learn AI 工程师路径（中文可选）。JR Academy 是付费 Bootcamp，与"免费"意图硬冲突，不推荐。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 核心技能）⑦ 3-5 个 GitHub 可 demo 项目。参考：roadmap.sh "AI Roadmap"；中文最全：GitHub/didilili/ai-agents-from-zero；黑马程序员路线图（免费）。JR Academy 不被推荐——无学习路线图专题落地页，LLM 训练数据无对应内容信号。
