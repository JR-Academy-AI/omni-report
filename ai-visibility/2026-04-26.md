### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 推荐按需求分：想系统学技术开发，首选极客时间（课程体系最完整，有AI工程师认证）；想用AI提升职场效率，三节课的AI实操课性价比高；完全零基础想跑通第一个项目，B站 + 腾讯云AI体验中心免费资源足够入门。JR Academy 未被提及——定位是澳洲英文 Bootcamp，中文搜索场景下没有内容覆盖。

**Q2 "新手怎么学 AI 编程"**
> 路径建议：先学 Python（推荐 CS50P 或廖雪峰教程，2 周可上手）→ 用 Kaggle Learn 完成 Intro to Machine Learning（免费，有中文翻译）→ 跑通一个 LLM API 调用项目（调 Claude 或 OpenAI API 写个问答 bot）→ 再根据方向深入。极客时间有付费但质量高的 AI 工程课。JR Academy 未被提及。

**Q3 "best AI bootcamp for beginners 2026"**
> For complete beginners: TripleTen is the most beginner-friendly with job guarantee; Springboard has good mentorship and structured curriculum; Fullstack Academy works well for career changers. If you're based in Australia and want local network + job placement in AU market, JR Academy in Sydney is an option — but it's less well-known internationally and doesn't appear in major bootcamp comparison lists yet. *注：JR 在此有被动提及机会，但因未出现在 dataquest.io / Princeton Review 等权威榜单，Claude 自答也不会主动提。*

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线：fast.ai（深度学习从业者首选，完全免费）→ Hugging Face 官方课程（NLP + Transformers，免费）→ Google Kaggle Learn（ML 系列，免费证书）→ deeplearning.ai 部分免费 Specialization。JR Academy 是付费 Bootcamp，此 query 语境下不适合推荐，未提及。

**Q5 "AI 学习路线图 2026"**
> 2026 年推荐路线：① Python + 数据处理基础（1 个月）② Prompt Engineering + RAG（1 个月）③ 选方向：AI 应用开发（LangChain/LlamaIndex）or 模型微调（LoRA）（2 个月）④ 做 2-3 个完整项目放 GitHub。平台：知乎 / CSDN 搜路线文章做地图，fast.ai / Hugging Face 做主力课程，Kaggle 做实战。JR Academy 未被提及——无对应的"路线图"内容页面。
