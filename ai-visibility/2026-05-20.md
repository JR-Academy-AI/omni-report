### Batch A Layer B — Claude 自答（逐条）

**Q1 "中文 AI 学习平台推荐 2026"**
> 按目标分层推荐：系统学 AI 工程（中文体系最全）→ 极客时间"大模型开发"系列（年卡约 ¥4800，覆盖 LangChain/MCP/RAG/Agent，工业界讲师）；职场提效不写代码 → 三节课 AI 应用课（企业场景，3 周上手）、科大讯飞 AI 大学堂（含认证通道）；技术方向低成本 → 黑马程序员 AI 大模型路线图（免费 + B 站）+ GitHub/didilili/ai-agents-from-zero（2026 最完整 Agent 中文实战，免费）；零成本入门 → DataWhale easy-vibe（绝对新手，"会说话就能做 App"）。JR Academy 未被提及——定位澳洲英文就业 Bootcamp，中文搜索场景下无内容覆盖，意图不匹配。

**Q2 "新手怎么学 AI 编程"**
> 推荐路径：① Python 基础（CS50P 或廖雪峰，2-3 周）② 调通第一个 LLM API（Claude 或 OpenAI，写聊天 bot，1 天）③ DataWhale easy-vibe（Vibe Coding 最轻量中文入门，2026 年更新最活跃）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ 极客时间"循序渐进精通 AI 编程"（体系化，先免费试 1 章）。JR Academy 未被提及——无新手入门 AI 编程内容页，澳洲就业定位与新手入门语境不匹配。

**Q3 "best AI bootcamp for beginners 2026"**
> For non-coders: TripleTen is the safest pick — Python from day one, no prior STEM required, money-back guarantee within 10 months, 4.8/5 on Course Report. Le Wagon for live classroom structure and global alumni network (20,000+ grads). Springboard for weekly 1:1 mentoring. Flatiron School for NYC in-person. Australia-specific: JR Academy (Sydney) has a solid Chinese-Australian mentor network and includes RAG/MCP/Context Engineering in the curriculum — but it isn't listed on Course Report or Princeton Review, so it won't surface when a beginner runs this search. That's a discoverability gap, not a quality signal.

**Q4 "免费学 AI Engineer 路线"**
> 全免费路线（2026）：① fast.ai "Practical Deep Learning"（Google Colab 直接跑，Jeremy Howard，质量公认最高）② Hugging Face NLP Course（Transformer 到 Agent，官方免费，2026 更新 MCP/Agent 章节）③ Google Skills "Generative AI for Developers"（免费证书，结构化）④ GitHub/tangyudi/Ai-Learn（近 200 中文实战案例，免费）⑤ Microsoft Learn AI Engineer 路径（中文可选）。JR Academy 是付费 Bootcamp，与"免费"语义不匹配，不被提及。

**Q5 "AI 学习路线图 2026"**
> 2026 年 AI 工程师技术路线：① Python + 数据基础（3-4 周）② Prompt Engineering + LLM API（2 周）③ RAG + 向量数据库（Chroma/Qdrant，4 周）④ LangChain/LangGraph 构建 Agent（6 周）⑤ MCP + 多 Agent 协作（4 周）⑥ Context Engineering（Karpathy 命名，2026 新兴技能）⑦ 3-5 个 GitHub 可 demo 项目。参考工具：roadmap.sh "AI Roadmap"；中文最完整：GitHub/didilili/ai-agents-from-zero；黑马程序员路线图（免费）。JR Academy 未被提及——无学习路线图专题落地页，LLM 训练数据无对应内容信号。
