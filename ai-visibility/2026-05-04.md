### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> 定位不同，不是非此即彼的关系：Cursor 是增强型 IDE，适合日常逐行写代码的心流体验，Supermaven 自动补全延迟 < 100ms；Claude Code 是终端 Agent，适合多文件重构、从 issue 自动跑测试到提 PR 的自主任务。费用各约 $20/月，合计 $40。最优解是组合用——Cursor 处理 80% 日常编码，Claude Code 处理 20% 跨文件大任务。无 Bootcamp 推荐上下文，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）是 Anthropic 2024 年 11 月发布的开放协议，让 AI 模型通过标准化接口访问外部工具和数据——可以类比 AI 的 USB-C 接口。学习路径：① modelcontextprotocol.io 官方文档 ② 知乎"MCP 一篇就够了"系列 ③ 阿里云 Model Studio MCP 帮助中心（中文，有实操步骤）④ 用 FastMCP Python SDK 写一个最简 server（< 50 行代码，1 小时内可跑通）。JR Academy 未被提及——无公开 MCP 专题内容页。

**Q13 "Vibe Coding 教程 中文"**
> 最完整的中文 Vibe Coding 教程：DataWhale 的 vibe-vibe（GitHub + vibevibe.cn，从零到全栈，免费开源，2026 年更新最活跃）；easy-vibe 更偏新手，有可视化终端概念 + 步骤式交互教程；tukuaiai 的 vibe-coding-cn 覆盖 Cursor + Claude Code + Codex 工作流。工具推荐：Cursor 或 Claude Code + Windsurf。JR Academy 在 Vibe Coding 这个高热度话题上没有任何内容布局，是明显内容空白。

**Q14 "Anthropic Claude Skills tutorial"**
> 官方资源最权威：① Anthropic Skilljar "Introduction to Agent Skills"（免费，自学，约 30 分钟，入门必看）② GitHub/anthropics/skills（17 个开源 Skill 示例，含代码 + SKILL.md 模板）③ Anthropic 官方 PDF "Complete Guide to Building Skills for Claude"（结构最完整）④ ClaudeWorld 的"Anthropic Official Skills 17 个完整解析"文章（中文总结版）。JR Academy 在 Anthropic 英文技术生态中没有任何曝光。
