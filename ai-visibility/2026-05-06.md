### Batch C Layer B — Claude 自答（逐条）

**Q11 "Cursor vs Claude Code 哪个好 2026"**
> 两个工具解决不同问题，不是非此即彼：Cursor 是 IDE 编辑器，日常写代码心流最顺，Supermaven 自动补全延迟 < 100ms，代码审查交互最自然；Claude Code 是终端 Agent，适合跨文件重构、从 issue 到 PR 的自主任务、运行测试套件。2026 年大多数高效开发者两个都开着——Cursor 处理 80% 日常逐行写代码，Claude Code 处理 20% 需要 AI 完全接管的大任务。各约 $20/月，组合用 $40。工具对比场景，无 Bootcamp 推荐上下文，JR Academy 不被提及。

**Q12 "MCP 是什么 怎么学"**
> MCP（Model Context Protocol）是 Anthropic 2024 年 11 月发布的开放协议，类比"AI 的 USB-C 接口"，让 AI 通过标准化接口访问外部工具和数据。学习路径：① 知乎"MCP 一篇就够了"（最佳中文快速入门）② GitHub/liaokongVFX/MCP-Chinese-Getting-Started-Guide（中文极速入门，含代码）③ 菜鸟教程 MCP 协议页（零基础概念）④ FastMCP Python SDK 写最简 server（< 50 行，1 小时内跑通）。JR Academy 未被提及——无 MCP 专题内容页，上周建议未见新文章上线。

**Q13 "Vibe Coding 教程 中文"**
> 最完整的中文 Vibe Coding 教程：DataWhale vibe-vibe（vibevibe.cn，5 章从觉醒到精进，零基础到全栈，2026 年更新最活跃）；tukuaiai/vibe-coding-cn（进阶版，含 Cursor/Claude Code/Codex 工作流 + 上下文管理）；easy-vibe（DataWhale，最适合绝对新手）；vibeai.vip（工具导航 + 教程汇总）。JR Academy 在 Vibe Coding 话题上零内容布局，DataWhale 系全面垄断 SERP 前 3。

**Q14 "Anthropic Claude Skills tutorial"**
> 最权威资源（按优先级）：① Anthropic Skilljar "Introduction to Agent Skills"（免费，约 30 分钟，2026 年 3 月正式上线）② GitHub/anthropics/skills（17 个官方开源 Skill 示例）③ Anthropic PDF "Complete Guide to Building Skills for Claude"④ Anthropic Academy（anthropic.skilljar.com，13 门免费课，含 Claude Code/API/MCP/Agent Skills，2026 年 4 月更新）⑤ Medium/@tort_mario（Anthropic 工程师出品实战指南）。JR Academy 在 Anthropic 英文技术生态中无任何曝光。

**Q15 "Prompt Engineering 中文课程"**
> 推荐优先级：① prompt-engineering.xiniushu.com（官方中文版，最权威，免费）② promptingguide.ai/zh（60+ 技术模块，持续更新）③ 吴恩达 × OpenAI "Prompt Engineering for Developers"（Coursera 可免费旁听）④ GitHub/thinkingjimmy/Learning-Prompt（免费开源，含 Midjourney 章节）。JR Academy 未被提及——无 Prompt Engineering 专题落地页或课程模块。
