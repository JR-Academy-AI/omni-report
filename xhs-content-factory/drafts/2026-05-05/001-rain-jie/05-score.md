## 各维度详细评分

### 1. 合规红线 ✅ 通过

详见 `04-review.md` 任务一。无任何违禁词命中，无引流违规，无承诺金钱 / offer。

### 2. AI 率（13/15）

预估 AI 率 12-18%。扣 2 分原因：
- "5 个错每个都『错 X：xxx / 修正：xxx』"是模板化结构（listicle 必要，但确实有节奏感）— 扣 1
- "卡在第几个" 收尾问句虽是真实疑问，但和"评论区告诉我"组合是常见 CTA 套路 — 扣 1

人味亮点：
- "我犯了 60 多次"自我承认 + 具体数字
- "we have stronger candidates 常常是 visa 问题不是技术" — 行业内人才会这样想

### 3. 标题钩子（12/15）

「AI 求职我犯过 5 个错 第三个最伤」

| 钩子要素 | 命中 |
|---|---|
| 数字（5 / 第三） | ✅ 双数字 |
| 痛点（最伤） | ✅ |
| 第一人称（我） | ✅ |
| ≤ 20 字（18 字） | ✅ |
| 反常识 / 悬念 | 🟡 部分（"第三个最伤"是悬念但不是反常识） |
| 含 1-2 个搜索关键词 | ✅ "AI 求职" |

扣 3 分原因：缺一个反常识强钩子（如"AI 求职 80 投 0 面 不是 AI 的错"会更钩）。

### 4. 开头 3 秒（8/10）

"去年我用 AI 写简历投了 80 家澳洲 IT 岗，0 个面试。
今年我重新投，3 个月拿了 7 个 onsite。"

✅ 第一句直接戳痛点（80 投 0 面）+ 给反差（今年 7 onsite）
🟡 但前两句独立成段，没有"对比冲击"那种节奏 — 可以合并为一句"去年 80 投 0 面，今年 3 个月 7 onsite，差别就在这 5 个错。"

扣 2 分。

### 5. 内容结构（14/15）

✅ 痛点（80 投 0 面）→ 方案（5 个错 + 修正）→ 证据（具体修正法）→ 行动（评论区互动） — 完整闭环
✅ 5 个分点，未超 5（恰好命中上限）
✅ 段落 ≤ 3 行（手机端可读）
✅ 多个具体案例 / 数字（80 / 0 / 7 / 60 / 80% / 100 字 DM）
✅ 无大段独白

扣 1 分：第 3 个错的"修正法"还可以更具体（"花 15 分钟看 LinkedIn 和发文"可以加 1 句"看什么"）。

### 6. 种草角度（13/15）

| 加分项 | 分值 |
|---|---|
| 第一人称使用经历（"我犯了 60 多次"） | ✅ 4 |
| 具体场景（"凌晨改简历"那种）| 🟡 3 — 故事 C 没用具体场景细节，干货风牺牲了部分场景感 |
| 踩过的坑 / 不完美点（80 投 0 面） | ✅ 3 |
| 对比（before/after 数据 + GPT 模板 vs 真实写法） | ✅ 3 |
| 给出明确"什么人适合 / 什么人别买" | 🟡 0 — 这版没明确说"哪类人适用这套方法 / 不适用" |

扣 2 分：缺"什么人适合"那一行。下次可加："如果你是 0 经验 + work visa，这 5 步顺序可能要调，最痛的是错 5"。

### 7. 配图（11/15） ⚠️ 基于 prompt 设计

**注**：5 张图都还是 prompt 阶段，未真实生成。本评分基于 prompt 设计意图。

| 项 | 分 | 评 |
|---|---|---|
| 数量 | 3/3 | 5 张（3-9 范围） |
| 首图钩子 | 3/4 | 3 版封面都有数字 + 反差，但 GPT-Image-2 中文字渲染稳定性 ~85%，员工要 2-3 次重生 |
| 图文一致 | 3/3 | 5 张图分别对应：5 个错 / 数据对比 / Bullet 改写 / Cover letter 示例 / CTA — 全部对应正文段落 |
| 比例统一 | 2/2 | 全 1024x1536（3:4） |
| 字号可读 | 0/2 | ⚠️ prompt 没强制最小字号 24px。生图前 prompt 需补"all text ≥ 30pt visual size" |
| 无水印 / 盗图 / AI 痕迹 | 0/1 | ⚠️ prompt 默认有"no watermarks, no AI artifacts"但员工生图后必须用 Read 工具看图复检 |

扣 4 分：字号最小要求 + AI 痕迹复检两项 prompt 没明示，员工执行时容易踩坑。

### 8. 互动钩子（7/10）

"5 个错里，第三个我犯了 60 多次。
真正打开局面是从 cover letter 第一段自己写开始的。
你现在卡在第几个？评论区告诉我，下篇拆怎么把 AI 用对。"

✅ 7/10 — 具体问句"卡在第几个"+ 限定选项（1/2/3/4/5）让评论门槛低
🟡 扣 3 分：CTA 是"评论区告诉我"通用版，没用 Rain姐人设特有的反问（如"哪个错你最不服？"会更挑战性，引出讨论）
