## 4. AI 行业动态影响课程方向

- **OpenAI GPT-5.5 发布（2026-04-23）** — 代号"Spud"，SWE-bench 88.7%，MMLU 92.4%，比 GPT-5.4 减少 60% 幻觉；OpenAI 称其为"最强 agentic coding 模型"，已向 Plus/Pro 用户推送，Codex 同步更新 — [CNBC 报道](https://www.cnbc.com/2026/04/23/openai-announces-latest-intelligence-model.html)。**对 JR 影响**：Vibe Coding Bootcamp 的"与 AI 结对编程"章节应更新为 GPT-5.5 + Codex 实操案例。
- **Anthropic Claude Design 上线** — 对标 Figma 的 AI 原型设计工具，[The Neuron 报道](https://www.theneuron.ai/explainer-articles/anthropic-just-dropped-a-32-page-playbook-for-teaching-claude-to-work-like-a-real-team-member/)；Claude Mythos（高级安全/编码模型）限量上线中。**对 JR 影响**：UX/设计 + AI 工作流内容有开课窗口。
- **MCP 达到 9700 万次安装（2026-03）** — Linux Foundation 旗下 Agentic AI Foundation 正式成立，锚定 MCP + OpenAI AGENTS.md + Block goose 框架 — [Latent Space 报道](https://www.latent.space/p/ainews-ai-engineer-europe-2026)。**对 JR 影响**：Scrimba 已上 37 分钟《Intro to MCP》，JR AI Engineer Bootcamp 无 MCP 模块是明显短板。
- **Google Gemma 4 发布** — 4 个开源变体，Apache 2.0 许可，针对边缘端 agentic AI — [Google Developers Blog](https://developers.googleblog.com/bring-state-of-the-art-agentic-skills-to-the-edge-with-gemma-4/)。**对 JR 影响**：开源模型部署课程方向值得跟进。
- **OpenClaw 开源** — GitHub 上 302K+ 星，增速为历史最快 — [AI News Digest](https://www.humai.blog/ai-news-trends-april-2026-complete-monthly-digest/)。**对 JR 影响**：可作为 Agent 实战课程案例项目。

## 6. 对 JR Academy 的建议

1. **本周内启动 MCP 模块**：Scrimba 已上线 37 分钟《Intro to MCP》，MCP 全球装机 9700 万次（2026-03），JR AI Engineer Bootcamp 无此内容。指派一位讲师本周录制"MCP 实战：用 5 个工具构建 Claude Agent 工作流"作为独立 Lesson，先上线再完善。
2. **更新 Vibe Coding 结对编程章节**：GPT-5.5 本周（04-23）刚发布，SWE-bench 88.7%，Codex 同步更新。将"AI 结对编程"章节的截图和演示切换至 GPT-5.5 界面，同时补充 Claude Code vs GPT-5.5 Codex 横评内容——这是本周最快的差异化窗口。
3. **在报名页加入岗位数据**：澳洲 AI Engineer 职位 662 个（LinkedIn）、全职 AI 岗 1,132 个（Glassdoor）、AI 技能薪资溢价 56%、LinkedIn 2026 澳洲 #1 Jobs on the Rise。把这 4 个数字放进 AI Engineer Bootcamp 报名页的"为什么学"区块，替换现有的泛化文案。
