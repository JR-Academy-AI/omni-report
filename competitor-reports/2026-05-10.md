## 4. AI 行业动态影响课程方向

本周三家 newsletter（The Rundown AI / TLDR AI / Ben's Bites）共同主线：Anthropic 商业地位逆转 + 开源中文模型集中爆发 + AI 监管进入政策通道。

- **Anthropic ARR 正式超越 OpenAI（2026-04 末确认）** — 月活 ARR $30B vs OpenAI $24B；Claude Code 营收 run-rate 突破 $25 亿，2026 年初以来翻倍；80% 收入来自企业客户 — [The AI Corner](https://www.the-ai-corner.com/p/anthropic-30b-arr-passed-openai-revenue-2026)。**对 JR 影响**：JR Vibe Coding Bootcamp 押注 Claude/Claude Code 方向的选择被市场验证，可在招生页直接引用"Claude Code 企业营收翻倍"作为为什么选 Claude 工具栈的理由。
- **Anthropic + OpenAI 双双推出企业 AI 服务合资公司（2026-05-04）** — 两家同周宣布企业 JV，加速企业客户 AI 部署 — [TechCrunch](https://techcrunch.com/2026/05/04/anthropic-and-openai-are-both-launching-joint-ventures-for-enterprise-ai-services/)。**对 JR 影响**：AI Engineer Bootcamp 的"企业 AI 部署"模块（已有）价值升温，建议更新该章节案例，引用这两个合资公司架构。
- **四家中国实验室 12 天内发布开源编程模型** — Z.ai GLM-5.1、MiniMax M2.7、Kimi K2.6、DeepSeek V4 相继亮相，综合编程能力相当于 Claude Opus 4.7 但成本不足其三分之一 — [llm-stats.com](https://llm-stats.com/llm-updates)。**对 JR 影响**：课程"模型选型"章节需新增这四款模型对比，特别是成本/性能 tradeoff——这是学员真实工作中最高频的决策场景。
- **白宫起草 AI 模型审查 EO（2026-05-07）** — 国家经济委员会主任 Kevin Hassett 称拟仿照 FDA 药物审批流程对新模型进行政府审查 — [MIT Sloan ME AI Dispatch](https://www.mitsloanme.com/article/ai-dispatch-1-7-may-2026/)。**对 JR 影响**：AI Engineer Bootcamp 增加"AI 合规与监管"章节的时机成熟，澳洲学员在企业端也面临类似监管压力。
- **DeepSeek-V4-Flash 发布（2026-05-04 完整版）** — 284B 参数 MoE 架构，每 token 只激活 13B，全 Tier-1 模型中激活参数最小，推理成本大幅降低 — [devFlokers](https://www.devflokers.com/blog/ai-news-may-2026-models-papers-open-source)。**对 JR 影响**：RAG 与推理成本优化章节可引用 DeepSeek-V4-Flash 作为低成本生产部署案例。
