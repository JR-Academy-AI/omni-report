## 1. 角色定位（先讲清楚边界）

**Bella 是 GEO 内容工厂的「视觉 / 视频 / 搬运」专精**——3 天/周节奏，对接 Serena 公众号长文 + 新媒体三人组爆款笔记，改编成视频 + 多平台分发。

### Bella 真正负责的

| 类型 | 内容 |
|---|---|
| 🎬 视频号 / 抖音 / B 站脚本改编 | 把 Serena 公众号长文 + 三人组爆款笔记拆成 30s-60s 视频脚本 |
| 📸 小红书原创笔记（视觉化）| 配图 + 短文案 + 发布 + 评论维护 |
| 🔄 跨平台资料搬运 | 把已发文章搬运到次要阵地（简书 / 头条号 / 大鱼号 / 一点资讯 / 博客园） |
| 🎨 配图素材生产 | Canva 模板批量产出，给 Serena / Summer / Lily / KIKI-CD 提供视觉素材 |

### Bella 不做

| 任务 | 谁做 | 为什么 |
|---|---|---|
| CSDN / 知乎专栏 / 掘金长文改写发布 | Serena | 长文不是 Bella 专精 |
| 公众号原创长文 + 排版 | Serena | 公众号 owner 是 Serena |
| 创始人 IP 写正文 | 创始人 | 个人观点 |
| 学员故事写正文 | 真实学员 | 必须真实 |
| jiangren.com.au 落地页编辑 | Claude / 工程师 | 代码任务 |
| GitHub repo | Claude / 工程师 | 代码任务 |
| 英文阵地（Medium / LinkedIn / Reddit）| 创始人 / 海外团队 | 语言 + 个人 IP |

## 2. Bella 的 6 大工作模块

### 模块 1：小红书原创笔记（8 个）

> Bella 做：素材抠图 / Canva 模板套用 / 配文 / 发布 + 评论维护

| G4-1 | 留学生澳洲求职日记 第 1 集：投简历日 | 6 图 + 600 字 | 0.5 天 |
| G4-2 | 第 2 集：第一面 | 6 图 | 0.5 天 |
| G4-3 | 第 3 集：技术面 | 6 图 | 0.5 天 |
| G4-4 | 第 4 集：拿 offer | 6 图 | 0.5 天 |
| G4-5 | 第 5 集：入职第一天 | 6 图 | 0.5 天 |
| C6 拆段 | 悉尼 AI Bootcamp 5 家对比 | 拆 5 个对比帖 | 1 天 |
| D3 拆段 | 澳洲 AI Engineer 薪资数据 | 拆 3 个薪资 infographic | 0.5 天 |

**小计：~4.5 工日 / 8 个笔记**

---

### 模块 2：视频号 / 抖音 / B 站脚本改编（每周 2-3 个脚本）

> 上游：Serena 公众号长文 + 新媒体三人组爆款笔记 → Bella 拆 30s-60s 视频脚本

| 上游内容 | 改编为 | 平台 | 节奏 |
|---|---|---|---|
| F1-F10 创始人 IP 文（公众号首发后）| 30s-60s 创始人观点视频 | 视频号（主品牌）+ 抖音 | 每周 1-2 个 |
| Serena 写的 C / D 类长文（横评 / 求职）| 60s 横评剪辑视频 | 视频号 + 抖音 + B 站 short | 每周 1 个 |
| Summer / Lily / KIKI-CD 爆款笔记 | 30s 高光剪辑 | 视频号 + 抖音 | 每周 1 个 |
| G6 学员一天 vlog（5 集）| 5-10 分钟 B 站长视频 | B 站 + YouTube | 全 4 周 1 集 |

**4 周总产出**：8-12 个短视频脚本 + 1 个 B 站长视频

**小计：~5 工日**

### 模块 3：跨平台资料搬运（次要阵地铺量）

|---|---|---|---|
| Serena 公众号长文（F / C / D 类）| 简书 + 头条号 + 大鱼号 + 一点资讯 | 改 10%（标题 + 开头）| 每篇 0.5 天 / 共 5-8 篇 |

| Serena CSDN 文 | 51CTO + 博客园（程序员二级阵地）| 改 10% | 每篇 0.3 天 / 共 3-5 篇 |

**4 周总产出**：12-16 篇跨平台搬运（相当于把 Serena 的产出量放大 2-3 倍曝光）

## 6. 协作流程（与 Serena / 三人组的链路）

```
Serena 写完公众号长文（F / C / D 类）
  ↓ 通知 Bella（飞书 / 微信）
Bella 24h 内：
  ① 拆 30s-60s 视频脚本 → 视频号 + 抖音 + B 站
  ② 跨平台搬运到简书 / 头条号 / 大鱼号 / 一点资讯（改 10-15%）
  ③ 提取金句卡片 → 小红书素材

新媒体三人组爆款笔记（小红书）
  ↓ 通知 Bella
Bella 24h 内：
  ① 改编 30s 高光视频 → 视频号 + 抖音

Bella 自己产出小红书原创笔记
  ↓ 反向给三人组当素材参考
```
