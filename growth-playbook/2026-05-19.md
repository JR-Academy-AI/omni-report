|---|---|
| **一句话描述** | 量子位峰会 5/20 榜单发布 2 小时内开赛：用至少 1 个榜单新工具，48 小时内做出可 demo 的项目；周日 5pm Discord 全场投票，Top 3 免费入 Vibe Coding Bootcamp |

| **本周做？** | ✅ 推荐本周启动（唯一窗口：5/20 榜单发布；5/22-5/23 刚好是周末，开发者有时间；下周热度消散无法复制） |

**落地步骤**：
1. **D-1（今天）**：搭落地页（1 页：规则 + 报名表单 + 奖励说明），在 Discord 开 `#榜单-48h-sprint` 频道；教研确认 1 名周末在线主持的讲师或助教

## 玩法 ② **「Claude Code vs Codex CLI 实测数据征集」**（内容钩子 + 社区 UGC）

| 字段 | 值 |
|---|---|
| **一句话描述** | 任何人把「写一个本地 CSV 清洗 CLI」用 Claude Code 和 Codex CLI 各做一遍，按标准表记录 4 个指标（完成时间 / token 消耗 / 代码行数 / 出错次数），提交 JR 表单；前 20 名提交者得 AI Engineer Bootcamp 工具链模块免费试听 + 工具选型决策卡 PDF |
| **灵感来源** | GitHub 年度开发者报告（数据驱动内容，第三方公信力）+ 知乎答题瓜分（有限名额竞争感） |
| **对应业务** | AI Engineer Bootcamp + Frontend Lab（拉新） |
| **目标用户** | 拉新——正在纠结「工具选型」的工程师 / 转型开发者，本周 Claude Code vs Codex 讨论是他们的主动搜索词 |
| **执行难度** | 🟢 小（Typeform 表单 + 评分 Excel 模板 + PDF 素材，D0 当天可上线） |
| **预估指标** | 1 周收集 80 份实测数据，发知乎「JR Claude Code vs Codex 实测数据集」获 1.5 万曝光；AI Engineer Bootcamp 试听预约新增 50 人，10% 转化付费 = 5 人 |
| **本周做？** | ✅ 推荐本周启动（Claude Code vs Codex CLI 对比讨论本周在知乎/B站/X热度峰值，下周可能被其他话题盖过；越快越有「第一份中文实测数据」的先发优势） |

**落地步骤**：
1. **D0（今天）**：制定标准任务说明（「用 Claude Code 和 Codex CLI 各实现同一需求：写一个本地 CSV 清洗 CLI，功能：去重 + 填空值 + 导出 cleaned.csv」）；出 Excel 评分模板（4 列：完成时间秒 / 估算 token 数 / 代码行数 / 出错次数 / 主观推荐分 1-5）
2. **D0**：Typeform 落地页上线（任务说明 + 上传评分表 + GitHub 链接），发到 JR Discord / B站社区 / 知乎「Claude Code 值得用吗」话题下
3. **D3**：够 30 份时出「中间数据摘要」帖（「我们已收到 30 份实测，Claude Code 平均用时 4.2 分钟 vs Codex 6.1 分钟……」），发知乎和 B站，继续引流
4. **D7**：足量数据后出完整分析帖——带图表 + 结论「哪种场景选哪个工具」，文末 CTA「想系统学 AI 工程工具链？→ AI Engineer Bootcamp 工具链模块免费试听」
5. **D7**：向所有 80 位提交者发邮件「感谢贡献数据 + 你的工具选型建议（基于你的数据位置）+ AI Engineer Bootcamp 早鸟席位」

## 玩法 ③ **「无硕博逆袭：21 天造出让面试官停下来的 AI GitHub」**（streak 打卡 + 求职裂变）

| 字段 | 值 |
|---|---|
| **一句话描述** | 针对知乎/小红书热议「本科 AI 生找不到 AI 岗」，JR 推出 21 天 Portfolio Sprint：每天 1 个具体 commit 任务（Week 1: RAG pipeline → Week 2: Agent 工具链 → Week 3: 性能测试 + deploy），每天在 Discord 贴当天 commit 截图打卡；完成 21 天者进 Job Referral 内推池 + 得 JR code review；邀 1 名同学同期打卡双方得 $150 AUD credit |

## 这周不推的玩法

- **「全线课程周末限时 10% off」** — 没有稀缺性叙事的随机折扣，不刺激决策。JR 客单价 $2,000-$4,000 AUD，报名决策本质是「我够不够格 + 我能不能完成」，不是「贵不贵」。折扣只会让观望者继续观望，等下次更大折扣；同时设定降价预期，后续全价销售更难。
- **「量子位峰会直播连麦」** — JR 在中国 AI 媒体圈的嘉宾资源和品牌认知不足以支撑与量子位联合直播的信任门槛；峰会当天主舞台已有百度/智谱等大厂背书，JR 强行入场效果稀释而非放大。更好的做法是「峰会结束后发声」（玩法①），而不是峰会期间蹭。
